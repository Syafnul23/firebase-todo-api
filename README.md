# 📝 REST API Todo List dengan FastAPI & Firebase

REST API sederhana untuk mengelola Todo List menggunakan FastAPI dan Firebase Firestore.

![FastAPI](https://img.shields.io/badge/FastAPI-005571?style=for-the-badge&logo=fastapi)
![Firebase](https://img.shields.io/badge/Firebase-039BE5?style=for-the-badge&logo=Firebase)
![Python](https://img.shields.io/badge/python-3670A0?style=for-the-badge&logo=python&logoColor=ffdd54)

## 🚀 Features

- ✅ CRUD Operations lengkap (Create, Read, Update, Delete)
- ✅ Firebase Firestore integration
- ✅ Auto-generated API documentation (Swagger UI)
- ✅ Input validation dengan Pydantic
- ✅ Error handling
- ✅ CORS enabled
- ✅ Filter todos by completion status
- ✅ Auto timestamps (created_at, updated_at)

## 📋 Prerequisites

- Python 3.8+
- Firebase Account
- pip (Python package manager)
